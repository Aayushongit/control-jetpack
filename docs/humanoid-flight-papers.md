# Notes

These papers cover the main research topics required to understand flying humanoid robots:

- Multibody dynamics
- Aerodynamic modeling
- Jet propulsion modeling
- Thrust allocation
- Model Predictive Control
- Reinforcement learning for flight

Studying these papers provides a strong foundation for designing simulators and controllers for jet-powered humanoid robots.
